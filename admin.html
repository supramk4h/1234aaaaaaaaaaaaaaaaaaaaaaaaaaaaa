Admin Requires Supabase

The admin panel needs a live Supabase connection to upload images and manage prompts.
To enable it, follow these steps:

Create a free project at supabase.com

Run the SQL schema from README.md

Create a Storage bucket called prompt-images

Open js/config.js and fill in your SUPABASE_URL and SUPABASE_ANON_KEY

Set USE_LOCAL_DATA: false in the same file

Create an admin user in Supabase → Authentication → Users
